Forensic Tool One - Tool Files Directory
------------------------------------------
This directory contains the binaries, libraries, and supporting files required for Forensic Tool One.
Before running the installation script, ensure that all necessary files are present in this folder.

Instructions:
- Place all executable binaries and related assets here.
- Do not modify the directory structure to ensure the installer can locate these files correctly.

For any issues related to missing or corrupted files, please refer to the documentation or contact support.
